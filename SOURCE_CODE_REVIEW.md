# Instructions for Source Code Reviewers

This extension is built using the [WXT framework](https://wxt.dev/) and this source archive was automatically generated by WXT as per their [documentation](https://wxt.dev/guide/essentials/publishing.html#firefox-addon-store). After unzipping, you can rebuild this extension using the following commands in a terminal inside the directory in which this document is located:

```
npm i
npm run zip:firefox
```

The `.output` directory then contains the built `media-control-X.Y.Z-firefox.zip` extension.
